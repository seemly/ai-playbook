# Playbooks

A compact, opinionated set of playbooks and coding guardrails for AI‑assisted development.

## What This Repo Contains
- `ai/PLAYBOOK.md`: the baseline workflow and AI rules.
- `ai/stacks/`: stack-specific overlays (e.g., WordPress).
- `ai/standards/`: language guardrails (PHP, JS/TS, CSS, WordPress).
- `templates/git/`: reusable templates for change workflows.
- `planning/`: future additions and proposals.

## How To Use
1. Start with `ai/PLAYBOOK.md` for the core workflow.
2. Add a stack overlay from `ai/stacks/` if applicable.
3. Use the language guardrails in `ai/standards/` when editing code.
4. Copy templates from `templates/git/` into working repos as needed.

## Suggested Entry Points
- Baseline rules: `ai/PLAYBOOK.md`
- Language standards index: `ai/standards/INDEX.md`
- WordPress overlay: `ai/stacks/wordpress/PLAYBOOK.md`

## Notes
- This repo is a reference, not an application.
- Keep updates small, scoped, and well‑documented.
